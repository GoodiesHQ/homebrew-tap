# Goodieshq Tap

## How do I install these formulae?

`brew install goodieshq/tap/<formula>`

Or `brew tap goodieshq/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "goodieshq/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
